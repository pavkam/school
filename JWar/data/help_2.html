JWar Strategy “Game”

 

  1. The Game. Some Notes.

The idea of the game is to destroy your opponents. At the start you have a single “Builder” vehicle which

you will place more conveniently near some resource fields and in an open spot. You will then build the

“Main Building” which will give you the possibility to build more units. Be sure to build more “Builders”

In the first stage of the game to be able to gather more resources faster. When you have a constant

Resource supply you will probably want to create “Killers” which are the war vehicles in this game.
